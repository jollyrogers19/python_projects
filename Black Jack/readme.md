# Blackjack

This project recreates the classic **Blackjack (21)** card game in a simple, interactive command-line format.
Players compete against the dealer, aiming to get as close to 21 as possible without going over.

---

##  Features

* Play against an automated dealer
* Handles Aces as 1 or 11 intelligently
* Allows player choices: **Hit** or **Stand**
* Automatically determines win, lose, or draw
* Uses lists, functions, and game logic for clean structure
* Beginner-friendly, simple, and fun

---

##  Concepts Practiced

* Functions and clean code structure
* Game loops and conditional logic
* List operations
* Randomization using `random` module
* Score calculation

---

##  How to Run

Run the following command inside the project folder:

```bash
python main.py
```

---

##  Description

The game deals two cards to both the player and the dealer.
Players choose whether to draw another card (**Hit**) or stop (**Stand**).
The dealer draws automatically based on standard Blackjack rules.

In the end, the program compares scores and announces the winner.
