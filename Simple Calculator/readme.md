# Calculator

This project is a simple command-line calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. The user can continue calculations using the previous result or start a new calculation anytime.

##  Features
- Supports +, -, *, and รท operations  
- Uses functions for clean and modular logic  
- Allows continuous calculations until the user exits  
- Interactive and beginner-friendly interface  

##  Concepts Practiced
- Functions and return values  
- Dictionaries for mapping operators  
- Loops and conditional logic  
- Working with user input
